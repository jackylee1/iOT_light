# iOT light bulb
* A lightbulb that are able to control using iOS application or web
* current stage planning to make localhost first

## Materials
* Light bulb (4.5V)
* nodeMCU
* USB Cable

## Tools
* Version control: Git
* Text editor:
	* NodeMCU: Arduino IDE
	* iOS App: Xcode
	* website: Atom / Sublime Text
	* iOT platform: ThingSpeak (if not localhost)
